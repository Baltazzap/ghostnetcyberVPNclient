# ChostNet Cybver VPN Client — чистый клиент на базе Hiddify

Это отдельный VPN-клиент на базе открытого кода Hiddify.

Изменено только брендирование:
- название приложения: `ChostNet Cybver VPN Client`;
- Android package: `ru.chostnet.cybvervpnclient`;
- иконки/цвета в стиле ChostNet;
- сборка Android APK и Windows x64 через GitHub Actions.

Функциональность Hiddify не ограничивалась: можно вставлять любые ключи и подписки, как в обычном Hiddify.

## Сборка

1. Загрузи содержимое архива в корень GitHub-репозитория.
2. Commit → Push.
3. Открой GitHub → Actions.
4. Запусти `Build Android APK` или `Build Windows EXE`.
5. Готовые файлы появятся в Artifacts.
